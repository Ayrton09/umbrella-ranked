A comprehensive, lightweight, and secure ranking system for Counter-Strike community servers.

---

## ✨ Features
- Kills, Deaths, KDR, Playtime, Weapon stats  
- Multi-language (EN / ES / PT / RU / CH)  
- SQLite & MySQL support  
- Anti-spam system  
- Autosave system  
- MultiColors chat  
- CS:GO & CS:S compatible  

---

## 📥 Installation

<details>
<summary>Click to expand</summary>

1. Download the latest files  
2. Place files:
3. addons/sourcemod/plugins/umbrella_ranked.smx
4. addons/sourcemod/translations/umbrella_ranked.phrases.txt
5. Restart server or change map  
6. Config auto-generated at: cfg/sourcemod/umbrella_ranked.cfg
